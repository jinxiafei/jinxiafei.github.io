拼图游戏

发表于 2018-04-27 | 分类于 java

游戏说明：
设计一款拼图游戏，要求点击图片按钮，实现图片按钮的移动，直到每一个按钮都到达指定位置游戏终止退出。
游戏设计思路：

  1. 准备一张图像文件；
  2. 创建N个按钮图标，每个按钮图标里面存入一张分割后的图片信息；
  3. 创建一个空白按钮用于和图标按钮交换位置，达到移动的效果；
  4. 乱序，将按钮图标乱序，完成游戏效果；
  5. 创建一个面板添加游戏开始和游戏结束按钮；
  6. 设计游戏窗口；
